Back Tutorial How it works? Users Manual

1. First of all, users will see the page displayed as the welcome page that will direct users to the home page.


2. At the upper left, users can see a menu button that will navigate users to About, Contact and Tutorial page. Meanwhile at the upper right, "+" button is to add new tasks.


3. This is the page where users will add their new tasks. Users are required to put the task name, set the task priority, and set a date to the task. For priority, there are 3 levels which are Low, Normal and High.


4. Tasks will be displayed as their name, priority and date that has been assigned. Once the task is done, users can check toggle as the task is done and will be displayed as completed.


5. Next, share buttons which users can share tasks to their friends. Task sharing is connected to their whatsapp contact. Lastly, the reorder button which users can reorder tasks as they want.


6. If users slide the task, there will be two options which users can edit or delete tasks. New changes will be displayed on the home page.
